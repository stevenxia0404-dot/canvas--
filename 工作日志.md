## 2026-06-07（星期六）

### 工作内容

**信号灯间距修复**
- 问题：三个信号灯（灰/琥珀/绿）间距视觉不统一，灰→黄间距大，黄→绿间距小
- 排查过程：`border-black`（排除）→ `shadow-inner`（排除）→ `space-x-2`
- 根因：CDN Tailwind v3 + Vite Tailwind v4 双版本下 `space-x-2` margin 叠加冲突，v3 用 `margin-left`、v4 用 `margin-right`，叠加不对称
- 方案：
  - CSS 圆 `div` → Unicode `●` 字符（无需 box-model，字体 kerning 天然均匀）
  - `space-x-2` → `gap-2`（CSS 原生 gap 无叠加）
  - 去掉 `border border-black`、`shadow-inner`

**原料图标 CSS Sprite 优化**
- 问题：工作台加载 40 张独立 WebP，HTTP/1.1 6 并发限制导致 iPad 上 1 分钟+ 才加载完
- 方案：40 张 WebP → 1 张 sprite 图 `icons.webp`（98KB，8×5 网格，cell 128×128）
- 实现：
  - 新增 `scripts/generate-sprite.mjs`（sharp 拼接脚本）
  - 新增 `src/sprite.json`（坐标映射：`{id}.col/.row`）
  - `App.jsx` 新增 `spriteStyle(id)` 辅助函数，百分比 `background-position/size`
  - 两处 `<img>` 替换为 `<div>` sprite 渲染（原料网格按钮 + 玻璃杯堆叠）
- 注意：`sprite.webp`（雪碧原料图）与输出文件 `sprite.webp` 重名，脚本输出改 `icons.webp`，原图从 git 恢复

**v2 合并上线**
- `v2` → `main` fast-forward merge（27 commits，无冲突）
- 前端部署：`cloud-agent-academy` Pages → `academy.boluomate.com`
- Worker API 部署：`canvas-academy-api` → `stevenxia0404.workers.dev`

**v2 更新说明 PDF**
- 需求：将 `v2更新说明.md` 制作为 Apple 官网风格的 PDF 文档，适时运用图标
- 工具链：Python + reportlab 4.5.1，微软雅黑字体（`msyh.ttc` / `msyhbd.ttc`）
