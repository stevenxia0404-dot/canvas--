# 云端学院（Cloud Agent Academy）项目介绍

## 一、项目概述

**云端学院**是一款面向航空机舱服务人员的**特调饮品配方考核 PWA 应用**。模拟特工任务场景，学员通过终端登录 → 任务大厅选择 → 调制台操作 → 毕业证书的完整流程，完成云端酒廊与客舱特调两大品类共 23 款饮品的配方训练与考核。

- **正式地址**：https://academy.boluomate.com
- **上线日期**：2026 年 5 月
- **当前版本**：v2.2（2026-06-28）
- **支持设备**：iPhone / iPad / Android 手机 & 平板 / 桌面浏览器
- **安装方式**：浏览器打开 → 添加至桌面 → 全屏独立运行（PWA）

---

## 二、核心玩法

```
终端登录  →  任务大厅  →  调制台  →  毕业证书
(身份验证)   (选配方)    (实操调制)   (成绩评定)
```

1. **终端登录**：特工风格打字机动画，输入姓名/工号/部门
2. **任务大厅**：23 款配方卡片，翻牌选任务，逐关解锁
3. **调制台**：从 43 种物料中选择载杯和原料，按序列加入 → MIX 混合 → 系统比对判定 PASS/FAIL
4. **毕业证书**：根据失败次数评定等级（ELITE / PROFICIENT / QUALIFIED）

## 四、数据库设计

### agents 表（学员记录）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | TEXT | 特工名称 |
| emp_id | TEXT | 员工编号 |
| dept | TEXT | 所属部门 |
| created_at | DATETIME | 登录时间（北京时间） |

### feedback 表（用户反馈）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | TEXT | 昵称 |
| message | TEXT | 反馈内容 |
| created_at | DATETIME | 提交时间（北京时间） |

### API 端点
| 方法 | 路径 | 鉴权 | 用途 |
|------|------|:--:|------|
| POST | /api/agents | Key | 写入学员记录 |
| GET | /api/agents | Key | 查询学员列表 |
| GET | /api/agents/export | Key | 导出 CSV |
| POST | /api/feedback | 公开 | 提交反馈 |
| GET | /api/feedback | Key | 查询反馈列表 |

## 七、PWA 能力

| 能力 | 实现 |
|------|------|
| 添加到桌面 | iOS `apple-mobile-web-app-capable` + Android `manifest.json` |
| 桌面图标 | 192×192 + 512×512（maskable） |
| 全屏运行 | `display: standalone`，无浏览器地址栏 |
| 离线缓存 | Service Worker cache-first 策略，静态资源离线可用 |
| 自动更新 | SW `skipWaiting` + 检测到新版本自动刷新 |
| 安全区适配 | `viewport-fit=cover` + CSS `env(safe-area-inset-*)` |
| 强制竖屏 | 手机/平板横屏显示"请竖屏"提示 + manifest `portrait-primary` |

## 八、数据流

```
用户操作（前端 localStorage）
       │
       ├── 登录 → POST /api/agents → D1 agents 表
       ├── 调制结果 → 前端 localStorage 存储
       ├── 反馈 → POST /api/feedback → D1 feedback 表
       └── 管理员三击 → DEBUG 面板
              ├── VIEW（查看全部学员/反馈）
              ├── DOWNLOAD（导出 agents.csv / feedback.csv）
              └── 外部管理页 worker-api.boluomate.com（输 Key 访问）
```

### 用户体验

**零门槛上手**
- 无需下载 App、无需注册账号，浏览器打开即用
- 特工终端风格界面，游戏化任务流程，学习像闯关
- 每完成一款饮品配方获得即时 PASS/FAIL 反馈，成就感强

**全场景覆盖**
- 飞航班等待间隙、休息室、家中——任何碎片时间都能打开练
- 断网照样运行，高空机舱内也能使用
- 桌面添加图标后和原生 App 体验一致，打开速度快

**操作直觉化**
- 43 种物料都有彩色图标，看一眼就认识
- 配方判定自动完成，学员只需关注操作步骤
- 错误即时提示，不用等教员批改

### 培训价值

**降本增效**
- 替代纸质配方手册，23 款饮品随时随地查阅练习
- 新人入职可自主完成基础训练，减少教员一对一辅导时间
- 配方更新无需重印材料，后台同步全员即时生效

**数据驱动**
- 自动记录每位学员的训练次数和通过率
- 管理员可导出数据，量化评估培训效果
- 发现高频失败配方，针对性强化教学

**体验升级**
- 相比传统"看手册→实操→教员打分"流程，学习路径更短
- 学员可反复练习不消耗实物物料，降低培训成本
- 游戏化机制提升学习积极性，不再是枯燥背书
